Kubernetesの~/.kube/configのcontextからnamespaecとサービスの一覧を出力する簡単なプログラムです。



## 実行結果例
```
##start
context [tab] namespace [tab] service-name
context [tab] namespace [tab] service-name
context [tab] namespace [tab] service-name
...
##finish
```
